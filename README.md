# Voices of Iran

A website designed to raise awareness about internet shutdowns, digital repression, and the human impact of nationwide blackouts in Iran.

## Purpose

This project aims to inform international audiences about the effects of internet restrictions on communication, education, businesses, and daily life.

## Features

- Interactive blackout timeline
- Economic impact visualizations
- Story-driven content sections
- Responsive design
- Modern user interface

## Technologies

- React
- Vite
- JavaScript
- CSS
- Vercel
